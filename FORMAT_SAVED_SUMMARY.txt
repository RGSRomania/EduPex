                           LESSON STRUCTURE
═══════════════════════════════════════════════════════════════════════════════
Each Lesson Contains:
✅ METADATA
   - Title: "LX - Lecția X" format
   - Summary: Lesson topic description
   - Subject: Matematica or Limba Română
   - Timestamps: Created and updated dates
✅ EDUCATIONAL CONTENT
   - Theory: 200-600 character explanation
   - Examples: 1-4 practical examples
   - Tips: 2-4 learning tips/advice
✅ ASSESSMENT
   - Question: Lesson-specific quiz question
   - Options: 4 multiple-choice answers
   - Explanations: For correct and incorrect answers

                         HOW TO USE THIS FORMAT
═══════════════════════════════════════════════════════════════════════════════
FOR NEW LESSONS:
1. Follow the template structure
2. Create lesson-specific question (IMPORTANT!)
3. Write 4 multiple-choice options
4. Provide explanations for all options
5. Test thoroughly before deployment
FOR UPDATES:
1. Maintain the structure
2. Keep title format consistent
3. Update question if content changes
4. Verify everything still matches
5. Test changes thoroughly
FOR BACKUPS:
Run: node backend/export_current_lessons.js
Creates JSON file with all lessons and questions
