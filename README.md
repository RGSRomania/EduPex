# EduPEX

## Overview

EduPex is an innovative educational application that adapts the DuoLingo model for Math and Romanian language learning, specifically tailored to the Romanian school curriculum. The platform uses gamification strategies to create an engaging, motivating, and effective learning experience for students in grades 5-8.

## Interface & User Experience

The EduPex app interface prominently features gamification patterns integrated into a streamlined, user-friendly learning platform designed to boost engagement, motivation, and retention:

- **Progress Tracking**: Experience points (XP), visual level bars, and hearts (lives) system
- **Achievement System**: Streaks for consecutive days of learning and badges for milestones
- **Daily Engagement**: Customizable goals with reminders and push notifications
- **Bite-Sized Learning**: 5–10 minute lessons optimized for frequent, short learning sessions
- **Community Features**: Leagues and leaderboards to foster healthy competition
- **Accessibility**: Freemium model with core content free and premium features available
- **Offline Learning**: Downloadable lessons for anytime practice

The UI maintains a visually appealing, minimalist style with clear typography and iconography, engaging animations, and sound effects that enhance the interactive feel without overwhelming users.
